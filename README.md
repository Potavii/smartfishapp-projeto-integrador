## Descrição

SmartFishApp é uma aplicação móvel inovadora desenvolvida para monitoramento e gestão inteligente de aquicultura. Este projeto demonstra habilidades avançadas em desenvolvimento de aplicações móveis cross-platform, utilizando tecnologias modernas como React Native, Expo e integração com Supabase para backend em tempo real. Ideal para portfólio, destaca competências em desenvolvimento full-stack mobile, design responsivo e integração de APIs.

## Funcionalidades

- **Navegação por Abas**: Interface intuitiva com abas para exploração e visualização de dados.
- **Integração com Supabase**: Backend robusto para armazenamento e sincronização de dados em tempo real.
- **Design Responsivo**: Adaptável a diferentes tamanhos de tela e dispositivos móveis.
- **Modal e Componentes Interativos**: Elementos UI avançados para melhor experiência do usuário.
- **Roteamento Baseado em Arquivos**: Estrutura organizada para desenvolvimento escalável.

## Tecnologias Utilizadas

- **React Native**: Framework principal para desenvolvimento de aplicações móveis nativas.
- **Expo**: Plataforma para desenvolvimento rápido, testes e distribuição de apps.
- **TypeScript**: Linguagem de programação com tipagem estática para maior robustez e manutenção.
- **Supabase**: Plataforma de backend como serviço para autenticação, banco de dados e APIs em tempo real.

## Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/smartfishapp.git
   cd smartfishapp
   ```

2. Instale as dependências:

   ```bash
   npm install
   ```

3. Inicie a aplicação:

   ```bash
   npx expo start
   ```

Na saída, você encontrará opções para abrir o app em:

- [Build de desenvolvimento](https://docs.expo.dev/develop/development-builds/introduction/)
- [Emulador Android](https://docs.expo.dev/workflow/android-studio-emulator/)
- [Simulador iOS](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), um sandbox limitado para testar o desenvolvimento de apps com Expo

## Uso

Você pode começar a desenvolver editando os arquivos dentro do diretório **app**. Este projeto utiliza [roteamento baseado em arquivos](https://docs.expo.dev/router/introduction) para uma estrutura organizada e escalável.

Para obter um projeto limpo, execute:

```bash
npm run reset-project
```

Este comando moverá o código inicial para o diretório **app-example** e criará um diretório **app** em branco onde você pode começar a desenvolver.
